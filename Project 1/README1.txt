Що робить програма:
Програма використовує перелічувані типи даних для обчислення та виводу завтрашньої дати відносно заданої в програмі. Система обробляє дати в різних календарях, григоріанському та юліанському.

Як запустити:
Збережіть файл project1-10.c собі на комп'ютер.
Для компіляції файлу введіть у консоль gcc project1-10.c -o Project - цей код створить робочий .exe файл(у вас має бути встановлений С компілятор в консолі).
Для активації програми введіть в консоль ./Project.
Після активації на консолі буде виведено: 
Terminal@PC :- $ gcc project1-10.c -o Project
Terminal@PC :- $ ./Project
